1370E - Binary Subsequence Rotation

題敘

Link

問做到對齊的最小次數。

895D - String Mark

題敘

Link
有兩個字串$a$，$b$， 問有多少$a$的排列滿足字典序大於$a$小於$b$。
輸出$\mod 10^9 + 7$。

想法

分開作小於$a$，小於$b$再相減，
用DP/排列組合就可以過。

細節

本題時限卡很緊，需要優化。
(建議可以用階乘反元素處理)

1336C - Kaavi and Magic Spell

題敘

Link
有一個空字串A，
依序把原字串S的前面放到A的頭或尾(不需要放完)，
有多少操作順序使得做出來的A有一個前綴是T。
輸出$\mod 998244353$。

想法

令$dp[i][j]$表示已經放了$[0..i]$，開頭在$j$，滿足條件的可能，
每次就只要考慮放前或放後兩種可能就好。

AC Code
